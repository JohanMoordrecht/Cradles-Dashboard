# Logistiek Dashboard - Cradle & Pallet Behoefte (Updatable!) Dit dashboard is nu **updatable**: upload een nieuwe Excel (of plak gekopieerde data) en alles wordt automatisch herberekend! ### Hoe update je de data? 1. Open je nieuwe Excel-bestand 2. Selecteer ALLE data (vanaf de koprij "Ordernummer" tot de laatste rij) 3. Kopieer (Ctrl+C) 4. Open dit dashboard (index.html) 5. Klik op "Nieuwe data plakken uit Excel" 6. Plak de gekopieerde data in het tekstvak 7.
